Limiti per il ripristino del sito di Azure:


|LIMITE IDENTIFICATORE|LIMITE PREDEFINITO|
|---|---|---|---|
|Numero di archivi per abbonamento|25|
|Numero di server per l'archivio di Azure|250|
|Numero di gruppi di protezione per archivio Azure|Nessun limite|
|Numero di piani di ripristino per Azure archivio|Nessun limite|
|Numero di server per gruppo protezione|Nessun limite|
|Numero di server per piano di ripristino|50|
