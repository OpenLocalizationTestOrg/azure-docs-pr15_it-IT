# <a name="how-to-view-and-manage-charts-of-data-about-your-application"></a>Come visualizzare e gestire grafici dei dati sull'applicazione
La sezione Dashboard dell'interfaccia utente gli utenti possono creare grafici personalizzati per ottimizzare l'ora, fornendo informazioni esatte che desiderano anziché cercare nella sezione analitica. Un dashboard Mostra le tendenze dell'applicazione e attiva utente conta per le diverse versioni dell'applicazione durante un determinato momento (ora/giorno/settimana/mese/personalizzato). Le tendenze visualizzate dipendono negli ultimi sette giorni.

### <a name="see-also"></a>Vedere anche
-  [Risoluzione dei problemi guida - Analitica][Link 21]

## <a name="dashboard"></a>Dashboard
- È possibile aggiungere grafici a un dashboard vuoto, scegliere l'azione "Invia a dashboard" dal menu a forma di ingranaggio di uno di essi nella sezione Analitica dell'interfaccia utente che si desidera visualizzare in un dashboard.
 
## <a name="-add-a-dashboard"></a>+ (Aggiungi un Dashboard)
- È possibile utilizzare la voce di menu della barra multifunzione "+" per creare un nuovo dashboard che verrà visualizzato sulla barra multifunzione.
 
## <a name="gear-menu"></a>Menu ingranaggio
- È possibile utilizzare il menu ingranaggio di un dashboard per eliminare, rinominare, condivisione, o pianificazione (ricezione di posta elettronica) del dashboard. È possibile utilizzare l'opzione di programmazione per definire la programmazione in cui si desidera ricevere il dashboard personalizzato tramite posta elettronica. È possibile pianificare riceverle giornaliera, settimanale o mensile. È possibile scegliere l'ora del giorno e il giorno della settimana e giorno del mese in cui si desidera ricevere la posta elettronica. L'indirizzo di posta elettronica e al fuso orario si basano sulle impostazioni configurate nella sezione Account personale dell'interfaccia utente.
