# <a name="azure-storage-client-tools"></a>Strumenti Client di archiviazione Azure

Gli utenti di spazio di archiviazione di Azure, spesso è necessario essere in grado di visualizzazione/interagire con i relativi dati utilizzando uno strumento di client di archiviazione Azure. Nelle tabelle seguenti sono elencati alcuni strumenti che consentono di eseguire questa operazione. Abbiamo inserire una "X" in ogni blocco se offre la possibilità di enumerare e/o accedere l'estrazione di dati. Nella tabella sono anche se gli strumenti è gratuita o meno. "Prova" indica che non esiste una versione di valutazione gratuita, ma il prodotto completo non è disponibile. "Y/N" indica che è disponibile una versione gratuitamente, mentre è disponibile per l'acquisto di una versione diversa.

Troverete solo uno snapshot degli strumenti client di Azure lo spazio di archiviazione disponibili. Questi strumenti possono continuare ad evolversi e aumento funzionalità. Se sono presenti correzioni o aggiornamenti, lasciare un commento sway.uservoice.com. Lo stesso avviene che se si è sicuri di strumenti che devono essere, si è soddisfatti per aggiungerli.
