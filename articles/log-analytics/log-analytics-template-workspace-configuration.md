# <a name="manage-log-analytics-using-azure-resource-manager-templates"></a>Gestire i Log Analitica utilizzando i modelli di Manager delle risorse di Azure

È possibile utilizzare [modelli di gestione di risorse Azure] (... / azure-resource-manager/resource-group-authoring-templates.md) per creare e configurare le aree di lavoro Log Analitica. Esempi di attività che è possibile eseguire con i modelli:

+ Creare un'area di lavoro
+ Aggiungere una soluzione
+ Creare ricerche salvate
+ Creare un gruppo di computer
+ Attivare la raccolta dei registri di IIS dal computer con installato l'agente di Windows
+ Raccogliere contatori da computer Linux e Windows
+ Raccogliere eventi da Registro di sistema in un computer Linux 
+ Raccogliere eventi da registri eventi di Windows
+ Raccogliere i registri eventi personalizzati
+ Aggiungere l'agente di analitica log una macchina virtuale Azure
+ Configurare analitica log indicizzare i dati raccolti mediante diagnostica Windows Azure


Questo articolo fornisce un modello di esempi che illustrano alcune delle configurazioni che è possibile eseguire da modelli.

## <a name="create-and-configure-a-log-analytics-workspace"></a>Creare e configurare un'area di lavoro Analitica Log

Esempio di modello seguente viene illustrato come:

1.  Creare un'area di lavoro
2.  Aggiungere le soluzioni nell'area di lavoro
3.  Creare ricerche salvate
4.  Creare un gruppo di computer
5.  Attivare la raccolta dei registri di IIS dal computer con installato l'agente di Windows
6.  Raccogliere contatori delle prestazioni del disco logico da computer Linux (% Inodes utilizzato; Gratuito megabyte; % Utilizzato uno spazio. Trasferimenti disco/sec; Lettura disco/sec; Scrittura su disco/secondo)
7.  Raccogliere gli eventi di registro di sistema da computer Linux
8.  Raccogliere eventi errori e avvisi il registro eventi applicazione dal computer Windows
9.  Raccogliere contatore delle prestazioni MB di memoria disponibili da computer Windows
10. Raccogliere un log personalizzato 
11. Raccogliere i registri di IIS e registri eventi di Windows di un account di archiviazione per diagnostica Windows Azure scritte

### <a name="deploying-the-sample-template"></a>Distribuire il modello di esempio

Per distribuire il modello di esempio:

1. Salvare l'esempio allegato in un file, ad esempio`azuredeploy.json` 
2. Modificare il modello per la configurazione desiderata
3. Usare PowerShell o riga di comando per distribuire il modello

#### <a name="powershell"></a>PowerShell

`New-AzureRmResourceGroupDeployment -Name <deployment-name> -ResourceGroupName <resource-group-name> -TemplateFile azuredeploy.json`

#### <a name="command-line"></a>Riga di comando

```
azure config mode arm
azure group deployment create <my-resource-group> <my-deployment-name> --TemplateFile azuredeploy.json
```
