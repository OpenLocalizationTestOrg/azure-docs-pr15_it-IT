# <a name="create-a-service-bus-namespace-using-the-azure-portal"></a>Creare uno spazio dei nomi Bus di servizio tramite il portale di Azure

Uno spazio quando un contenitore comune per tutti i componenti di messaggistica. Più code e argomenti possono trovarsi in un unico spazio dei nomi e gli spazi dei nomi vengono spesso utilizzate come contenitori di applicazioni. Esistono diversi modi per creare uno spazio dei nomi Bus di servizio.

1.  Portale Azure (in questo articolo)

2.  [Modelli di Manager delle risorse][create-namespace-using-arm]
