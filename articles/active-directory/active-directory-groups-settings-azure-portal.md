# <a name="manage-the-settings-for-a-group-in-azure-active-directory-preview"></a>Gestire le impostazioni per un gruppo in anteprima di Azure Active Directory

In questo articolo viene spiegato come modificare le impostazioni per un gruppo in anteprima di Azure Active Directory (Azure Active Directory). [Che cos'è nel riquadro di anteprima?](active-directory-preview-explainer.md)

## <a name="how-do-i-find-and-change-the-settings"></a>Come trovare e modificare le impostazioni

1.  Accedere al [portale di Azure](https://portal.azure.com) con un account è un amministratore globale per la directory.

2.  Selezionare **altri servizi**, immettere **gli utenti e gruppi** nella casella di testo e quindi premere **INVIO**.

  ![Gestione degli utenti di apertura](./media/active-directory-groups-settings-azure-portal/search-user-management.png)

3.  Scegliere **tutti i gruppi**e **l'utenti e gruppi** .

  ![Apertura e il gruppi](./media/active-directory-groups-settings-azure-portal/view-groups-blade.png)

4. Scegliere un gruppo e **l'utenti e gruppi: tutti i gruppi** .

5. Nel * *gruppo - *nomegruppo* ** blade, selezionare **proprietà * *.

  ![Apertura e il proprietà](./media/active-directory-groups-settings-azure-portal/select-group-properties.png)

6. Al termine della modifica delle proprietà per il gruppo, selezionare **Salva**.    

  ![Salvare le modifiche di proprietà](./media/active-directory-groups-settings-azure-portal/save-group-properties.png)
